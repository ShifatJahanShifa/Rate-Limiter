
# Rate-Limiter

I have implemented a rate-limiter using token-bucket algorithm.
## Technology

```
Backend : Node.js, Express.js

```



## Installation

clone the project:  

```bash
  https://github.com/ShifatJahanShifa/Rate-Limiter.git
```

## Run Locally

```bash
   cd server
   npm install
   node index.js
```
